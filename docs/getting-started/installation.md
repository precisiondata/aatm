# Installation

Install the package in your environment.

```bash
pip install aatm
```

If you want to build from the source, clone the repository and install it locally:

=== "pip"

    ```bash
    git clone https://github.com/precisiondata/aatm.git
    pip install -e . 
    ```

=== "uv"

    ```bash
    git clone https://github.com/precisiondata/aatm.git
    uv sync
    ```
